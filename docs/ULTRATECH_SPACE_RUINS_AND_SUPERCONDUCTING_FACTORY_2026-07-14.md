# UltraTech 开发日志 - 2026-07-14

## 太空残骸与遗迹探索

本次新增六个星球维度的遗迹探索内容，遗迹在既有地形上生成，不改写星球地表生成器。

| 星球 | 遗迹主题 | 核心发现 |
| --- | --- | --- |
| 月球 | 废弃前哨站、坠毁探测器、古代观测站 | 月球科技碎片、钛合金残片、研究数据碎片 |
| 火星 | 废弃基地、干涸运河、气象站 | 火星科技核心、稀有合金残片、地质样本 |
| 水星 | 采矿设施、热力发电残骸 | 耐热合金部件、太阳能碎片、辐射屏蔽板 |
| 谷神星 | 冰下实验室、钻探平台 | 冰层钻探核心、水冰样本、碳质陨石碎片 |
| 木卫一 | 火山观测站、硫磺采集设施 | 耐腐蚀合金、硫磺结晶、火山记录仪 |
| 冥王星 | 深空通信站、极地研究设施 | 深空通信零件、冷冻样本、冥王石晶体 |

- 每处遗迹由地表暴露结构、地下延伸、入口、可选障碍与中心核心组成。
- 遗迹最小间距和生成密度由星球类型决定，并会贴合现有地表高度。
- `ArchaeologicalScannerItem` 使用 FE 搜索附近遗迹，并通过 `RuinLocator` 返回目标位置。
- `RuinsDetectorBlockEntity` 在遗迹附近完成标记；`RuinExplorationSavedData` 保存已发现与已探索地点。
- `ArtifactAnalysisTerminalBlockEntity` 对未分析文物执行耗时分析，将结果接入研究数据和后续配方解锁链。
- 风化金属、风化玻璃、遗迹钢材、遗迹电路、遗迹核心与遗迹锁门构成可探索结构材料；专用遗迹挖掘工具用于保留特殊掉落。

## 超导工厂

阶段 22 之后的超导工厂已按五座独立多方块建筑落地，并通过低温管路、磁场传导器与超导汇流排组成闭环。

```text
低温恒温核心
  -> 液氮 / 液氦
超导制备区
  -> 超导线圈
磁场生成环
  -> 强磁场
磁约束容器
  -> 约束等离子体
磁流体发电阵列
  -> FE 回馈全厂
```

| 建筑 | 尺寸 | 主要职责 |
| --- | --- | --- |
| 低温恒温核心 | 8 x 8 x 6 | 维持低温工质并向产线提供冷却能力 |
| 超导制备区 | 三条多方块产线 | 消耗冷却介质与原材料，制造超导线圈 |
| 磁场生成环 | 10 x 10 x 4 | 消耗线圈与 FE，输出强磁场 |
| 磁约束容器 | 10 x 8 x 6 | 验证等离子体磁约束并维持约束状态 |
| 磁流体发电阵列 | 12 x 6 x 5，四通道 | 将等离子体和磁场转换为 FE |

### 结构和端口

- 五类控制器共用多方块扫描框架，结构定义由 DataGen 输出至 `data/ultratech/multiblock_structures/superconducting_factory/`。
- 建筑使用 8 级结构机壳、强化机壳、透风口、齿轮箱、电路板和输入/输出仓室体系。
- `cryogenic_pipe` 用于低温介质连接，`magnetic_field_conductor` 传递磁场状态，`superconducting_busbar` 提供超导能源链路。
- 控制器检查结构完整性、相邻端口、冷却介质、线圈、等离子体和 FE；任一关键链路不足时停止对应产出。

### 运行闭环

1. 低温恒温核心提供液氮或液氦。
2. 超导制备区消耗冷却介质和材料，产出超导线圈。
3. 磁场生成环消耗超导线圈和 FE，生成强磁场。
4. 磁约束容器消耗强磁场和等离子体，建立约束状态。
5. 磁流体发电阵列消耗约束等离子体与磁场，输出 FE。
6. FE 通过汇流排回馈全厂，其中一部分持续维持低温和磁场系统。

## 主要实现位置

- `com.halfsword.ultratech.ruins`: 遗迹选址、生成、持久化与世界事件。
- `SuperconductingFactoryControllerBlockEntity`: 工厂控制器、结构状态、端口验证与生产循环。
- `SuperconductingLinkBlockEntity`: 低温、磁场与超导网络节点。
- `ModSuperconductingFactoryStructureProvider`: 五座建筑的 DataGen 结构输出。
- 对应的 Block、BlockEntity、Menu、Screen、注册、语言、模型、战利品与配方已随 DataGen 资源闭合。
