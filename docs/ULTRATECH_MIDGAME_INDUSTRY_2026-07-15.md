# UltraTech 中后期工业基础实装记录

**日期：** 2026-07-15
**目标版本：** Forge 1.20.1 / Forge 47.4.4 / Java 17
**构建：** UltraTech 2.0.0 Demo

## 概览

本轮将阶段 10-22 的中后期工业基础正式接入 UltraTech 的工业过程框架。新增内容不是独立的展示条目：每台机器均已注册为可放置方块，拥有 FE、物品与流体能力，生成对应配方、语言、模型、战利品表和工业过程贴图。

本次新增 **132 台工业机器**，并补齐核燃料、工程陶瓷、纺织复材、表面强化、粉末冶金、造纸包装、独立硅基半导体、生物化工、制冷、工业气体、聚变燃料、月壤氦-3 和超级合金的中间物料链。

## 特殊运行逻辑

- 核燃料循环使用独立 `RadiologicalProcessMachineBlockEntity`。GUI 会显示模拟辐射剂量、屏蔽完整度和安全/警告/危险状态。
- 高温陶瓷烧结炉、陶瓷 HIP、工业热泵、月球采矿平台、VIM、VAR 和 ESR 使用专用多方块校验；结构定义由 DataGen 输出。
- 独立硅基路线保留现有硅工业设备，同时补上酸洗、气相前驱体、研抛、光刻、刻蚀、离子注入、CVD、切割和封装的后段流程。
- 聚变燃料与氦-3 产线通过氘、氚、DT 混合物和低温氦气介质与阶段 20-24 聚变设备联动。
