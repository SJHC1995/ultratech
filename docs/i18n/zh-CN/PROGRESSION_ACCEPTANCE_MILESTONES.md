# UltraTech 进度手动验收

这六个世界是发布门槛，不用于替代任务书。每条路线都必须从全新的
资源条件开始实际游玩；在 JEI 中用 R/U 导航检查；完成存档、重载，
并在资源重新加载后再次检查。

| 里程碑 | 阶段 | 目标时长 | 手动路线 |
|---|---:|---:|---|
| M1 | 1-5 | 35 小时 | 原版资源 → 稳定的 SU → 标准机械部件 |
| M2 | 6-10 | 60 小时 | 蒸汽 → 乙醇 → 压力物流 → 连续加工 |
| M3 | 11-15 | 75 小时 | FE 电网 → 化学中间体 → 自动化装配 → 研究终端 |
| M4 | 16-20 | 80 小时 | 低温 → 高纯度 → 并行加工 → 回收 |
| M5 | 21-25 | 160 小时 | 裂变/聚变 → 超导 → 高磁场基础设施 |
| M6 | 26-30 | 290 小时 | 算力 → 量子/宇宙链 → 巨构建筑 → 旧系统改造 |

每个里程碑都应记录世界种子、已测试的配方 ID、所需研究、可用的替代
输入路线、回收出口、JEI 分类结果、存档/重载结果，以及运行时配方或
JEI 覆盖审计账本中仍存在的警告。

进入每个验收世界后，以管理员身份执行 `/utaudit`。该命令使用实时
`RecipeManager`；它是在游戏内对应生成 manifest 检查的验证方式。
