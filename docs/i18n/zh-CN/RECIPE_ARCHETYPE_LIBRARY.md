# UltraTech 配方原型库

本库是 UltraTech 原创设计资料。它记录的是配方族的契约，而不是复制
任何第三方配方、标识符、资产或文本。

| 原型 | 阶段 | 批量契约 | 必需的回收出口 |
|---|---:|---|---|
| 原料预处理 | 1-4 | 1 单位原料 → 2 单位预处理产物 | 尾矿用于建材或再粉碎 |
| 矿物选矿 | 3-10 | 2 单位粉碎矿 + 250 mB 水 → 2 单位精矿 + 尾矿 | 陶瓷填料或回收 |
| 金属冶炼 | 4-13 | 2 单位精矿 → 2 单位金属 + 炉渣 | 再熔炼或耐火填料 |
| 标准件 | 4-15 | 板材 + 杆材 + 紧固件 → 1-4 个部件 | 废料再熔炼 |
| 蒸汽循环 | 5-12 | 处理水 + 燃料 → 蒸汽 + 冷凝水 | 冷凝水回流 |
| 生物精炼 | 6-12 | 生物质 + 水 → 燃料中间体 | 纸浆、热能、土壤改良料或炭 |
| 石油化工裂解 | 12-18 | 800 mB 馏分 → 320-480 mB 产物 + 尾气 | 洗涤处理或发电 |
| 氯碱—聚合物 | 12-18 | 300 mB + 300 mB → 500 mB 单体 | 聚合物回磨或中和 |
| 工业气体合成 | 14-20 | 150 mB N2 + 450 mB H2 → 300 mB 氨 | 催化剂再生 |
| 精密材料 | 15-22 | 2 路高纯原料 + 基材 → 元件 | 不合格品回收 |
| 低温超导 | 20-29 | 4 个线圈 + 250 mB 液氦 → 场部件 | 线圈与低温工质回收 |
| 核燃料循环 | 20-25 | 浓缩链 → 燃料组件 | MOX 或玻璃固化废物 |
| 多方块启动 | 21-30 | 结构部件 + 首次装料 → 基础设施 | 维护物料回收 |
| 终局材料 | 26-30 | 大批量生产 → 稀缺终端材料 | 异常废物回收 |
| 循环回收 | 4-30 | 次级路线的回收量低于主路线 | 明确的下一出口 |

`RecipeArchetype` 会在 manifest 中为每个由 DataGen 生成或手工编写的
UltraTech 配方分类。`IndustrialRoute` 则把该配方族关联到玩家可见的
路线、替代来源、回收出口、JEI 分类与机器状态图标。

DataGen 输出：

- `data/ultratech/progression_audit/recipe_archetype_coverage.json`
- `data/ultratech/industrial_routes/*.json`

DataGen 后运行 `scripts/verify-recipe-archetypes.ps1`。某个专用原型若
没有任何已分类配方，就是发布阻断项；顾问级发现必须在对应里程碑验收
前解决。
