# UltraTech Demo 2.0.0 发布说明

发布日期：2026-07-15

## 本次内容

Demo 2.0.0 汇集了当前 UltraTech 的电化学工业实现，并保留既有的工业、聚变、航天与多方块系统。

- 新增水电解器、电镀槽、阳极氧化槽、电渗析器、有机电合成反应器与电镀层检测仪。
- 扩充氯碱、电解精炼、电镀、电渗析和有机电合成的工业过程配方。
- 新增饱和盐水、烧碱溶液、金属电镀液、纯化水等电化学流体及配套物品。
- 熔盐电解炉改为独立的 3x3x3 多方块结构，要求精英级外壳、透风口和电路板。
- 调整关键电解设备的 FE 需求，使氯碱、电解铜、电积锌和熔盐电解的能耗与阶段定位一致。

## 安装

运行环境：

- Minecraft 1.20.1
- Forge 47.4.4
- Java 17
- GeckoLib Forge 4.8.4

下载并解压 Demo 压缩包后，将 `mods` 目录中的 `ultratech-2.0.0-demo-2026-07-15.jar` 与 `geckolib-forge-1.20.1-4.8.4.jar` 一同放入同一 Forge 实例的 `mods` 目录。

JEI、The One Probe 和 Patchouli 仍是可选依赖。
