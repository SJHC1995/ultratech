# UltraTech 2026-07-12 聚变阶段 Demo 发布与规划

## 本次落地

- 产出 Forge 1.20.1 Demo 构建，运行环境为 Forge 47.4.4 与 Java 17。
- GeckoLib Forge 4.8.4 为强制运行时依赖，Demo JAR 不内置该第三方模组。
- 阶段 19-24 的熔盐堆、氘聚变堆、氦聚变堆、质子加速器、托卡马克环和激光聚变室已接入独立 BlockEntity 逻辑、FE 行为、物品/流体输入验证和 GUI 指标。

## 关键实现说明

- 氘聚变堆的 100000 FE 点火使用相邻 FE 网络的突发事务抽取，避免受普通机器内部 FE 缓冲上限影响。
- 托卡马克环使用独立氘/氚罐，并要求相邻超导薄膜沉积系统作为当前框架中的超导支撑。
- 仓库既有氦-3供应形式为 `helium3_capsule`，氦聚变堆直接消费该胶囊并在机内按 mB 保存，未新建会导致批量容器资源缺口的全局氦-3流体。
