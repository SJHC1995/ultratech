# UltraTech Documentation Archive

This directory contains the project design documents, milestone roadmaps, inventory snapshots, and local historical snapshots used by the static blog.

- Browse the rendered archive from `../updates.html`.
- Read a document through `../document.html?file=docs/<file-name>.md`.
- Files with `_LOCAL_SNAPSHOT` preserve an older local copy when it differs from the version that was already published on the site.
