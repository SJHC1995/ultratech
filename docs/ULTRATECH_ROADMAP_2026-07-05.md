# UltraTech 2026-07-05 航天、算力与装配系统规划记录

本文整理 2026-07-05 当天围绕航天系统、星图、月球天空、多环境组装智能部署机器人集群、太空算力站、探测器/中继装配终端、JEI 配方、GUI 排版、本地化和构建稳定性提出并处理的内容。它同时作为下一轮进游戏验收清单。

## 今日核心目标

- 航天与星图体验继续向“可玩、可理解、可信”收拢：自动航行、目标星体、返航贴图、星球锁定和探测前置都要有一致规则。
- 算力系统统一用“网络缓存 / 算力请求 / 实际写入”的语义，不在普通 GUI 明面上暴露 120% 这类调参字样。
- 新增装配体系必须是真实消耗：背包材料、终端 FE、玩家绑定网络缓存都通过服务端校验后才产出装配包。
- 多环境组装智能部署机器人集群监视终端继续按参考图打磨：左侧大画布，右侧第一信息块突出电力和算力，滑动条可拖动，信息不压边界。
- 今日规划 Markdown 与 Gitee Pages 首页同步更新，作为可追踪博客式记录。

## 今日已完成

### 装配终端与装配包

- 新增并接入三类装配终端：太空算力站装配终端、行星探测器装配终端、中继信号站装配终端。
- 装配 GUI 使用独立材料清单界面：材料图标不再渲染数量，数量只在 tooltip 中显示。
- 服务端装配逻辑改为事务式顺序：
  1. 读取对应等级配方。
  2. 在背包快照中完整模拟材料消耗。
  3. 检查装配终端内 FE 是否足够。
  4. 检查玩家绑定网络缓存是否足够；不足时才发起算力请求。
  5. 全部满足后，统一扣除网络缓存、终端 FE 和材料，并发放装配包或探测器/中继产物。
- 修复“终端没电也向网络请求算力”的问题：现在 FE 不足时直接提示，不会把算力请求送入网络。
- 装配终端 FE 能量通过 Forge 能量能力和 BigEnergy 能力暴露，并持久化保存；外部充电导致 FE 变化时会标记方块实体已变更。
- 装配 GUI 增加“终端电量：当前 / 需求”行，方便直接看出是否真的有电、是否会被扣除。

### JEI 与配方

- 太空算力站、行星探测器、中继信号站三套 7 级装配配方已接入 JEI 分类。
- 行星探测器和中继信号站按材料清单生成 7 级装配配方。
- 缺失的装配组件物品、模型、64x64 贴图和语言键值已补齐，包括线缆束、密封件、散热器、天线、收发器、太阳能翼、推进剂罐等。

### 中文键值与显示规则

- 将空间工业终端中的“网络算力缓存”统一改为“网络缓存”。
- 将空间工业终端的“算力请求：%s/t”改为“算力请求：%s”，避免把一次性装配需求误显示成每 tick 数值。
- 修复新装配终端、JEI 分类、线缆束、密封件、散热器等一批 `????` 中文键值。
- 修复装配终端 FE 不足提示键值。

### 多环境组装机器人集群与监视终端

- 新增 7 级“多环境组装机器人集群”物品，并逐步替换旧的单体机器人表述。
- 多环境组装智能部署机器人集群监视终端使用可拖动滑动条切换星体。
- 监视终端左侧以高级星图/星空画布为总览基础，右侧信息块继续按参考图靠近：电力、算力、机器人群、材料、环境、通信、阶段、操作和历史记录。
- 已部署/有中继的平台才应显示对应监视内容；未部署星球不显示伪动画，这是后续进游戏验收重点。

### 航天与月球天空

- 月球维度天空效果接入独立渲染，不再沿用主世界天体。
- 月球返航和接近月球时，月球图标不再依赖缺失贴图，避免紫黑块。
- 航行动画的目标星体、进度单向推进、进入太空维度后预加载目标附近 16x16 区块仍列为后续重点打磨项。

## 当前验收重点

| 优先级 | 项目 | 验收标准 |
| --- | --- | --- |
| P0 | 装配终端服务端校验 | 背包材料不足时不产出、不扣 FE、不请求算力；FE 不足时不产出、不请求算力；网络缓存不足且材料/FE 已满足时才发起算力请求 |
| P0 | 装配终端 FE 显示 | GUI 能看到终端已存 FE 与当前配方 FE 需求 |
| P0 | JEI 装配配方 | 三类装配配方都能在 JEI 中显示，材料数量只在 tooltip 或配方文字中出现，不压材料图标 |
| P0 | 中文键值 | 新装配终端、探测器/中继装配、组件物品、JEI 分类不再显示 `????` 或原始键名 |
| P0 | 算力网络 | 合并 10 tick 请求，网络缓存为合并请求乘冗余，实际写入保持 0.94 到 0.99 的稳定浮动 |
| P0 | 探测器与中继 | 关闭游戏后探测器线程、中继平台和静默记录不丢失，支持同一玩家多个探测线程 |
| P0 | 导航台 | 未探明星球禁止写入星图坐标；已探明星球可写入，并显示正确星体名称 |
| P0 | 航行与降落 | 进入太空维度后自动航行，进度不回跳，目标附近 16x16 区块预加载，降落 Y 坐标按目标规则执行 |
| P1 | 火箭实体 | 可放置、可骑乘、可破坏、有贴图，放下不崩进程 |
| P1 | 火星/水星地貌 | 地貌实际生成，矿物不串星球，不出现月球矿跑到火星/水星的问题 |
