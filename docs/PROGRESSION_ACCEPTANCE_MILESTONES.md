# UltraTech Manual Progression Acceptance

These six worlds are release gates, not a task-book replacement. Every route
must be played from fresh resources, checked in JEI with R/U navigation, saved,
reloaded, and then checked after a resource reload.

| Milestone | Stages | Target | Manual route |
|---|---:|---:|---|
| M1 | 1-5 | 35h | Vanilla resources -> stable SU -> standard mechanical parts |
| M2 | 6-10 | 60h | Steam -> ethanol -> pressure logistics -> continuous processing |
| M3 | 11-15 | 75h | FE grid -> chemical intermediate -> automated assembly -> research terminal |
| M4 | 16-20 | 80h | Low temperature -> high purity -> parallel processing -> recovery |
| M5 | 21-25 | 160h | Fission/fusion -> superconductivity -> high-field infrastructure |
| M6 | 26-30 | 290h | Compute -> quantum/cosmic chain -> megastructure -> legacy retrofit |

For every milestone record the world seed, tested recipe IDs, required
research, available alternative input route, recovery outlet, JEI category
result, save/reload result, and any remaining warning from the runtime recipe
or JEI coverage ledgers.

Run `/utaudit` as an operator after entering each acceptance world. The command
uses the live `RecipeManager`; it is the in-game verification counterpart to
the generated manifest checks.
