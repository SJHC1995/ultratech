UltraTexture Studio

面向 Minecraft 资源包、批量拼图和 MC GUI 草图的本地图像工作台。程序不会托管或赠送上游图像服务；使用者在本机设置自己的 API Key、Base URL、模型、代理与 User-Agent。

  • 文生图与图生图兼容 OpenAI 风格图像接口，也提供精简 JSON 兼容模式。
  • MC 像素贴图支持 16×16、32×32、64×64 目标尺寸，并使用最近邻输出 PNG。
  • 批量与拼图一次处理最多 25 条描述，支持 CSV / JSON 导入、网格与边框。
  • 本地 MCP Bridge提供 /generate、/generate_texture、/generate_batch、/generate_grid 与 /docs。
